# Escampe
![Escampe](https://github.com/Poulpy/escampe/blob/master/escampe.png?raw=true)




## Clone the project

```bash
git clone https://github.com/Poulpy/Escampe.git
```

## Dependencies

- libsdl1.2-dev
- libsdl-ttf2.0-dev

## Compile
```bash
make
```

## Execute

```bash
./bin/main
```
